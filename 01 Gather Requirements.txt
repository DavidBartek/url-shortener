Functional Requirements
(use cases, actions for users to perform)

1. URL shortening - pass in long URL and receive shortened version
	-get unique URL - 1-1 mapping
2. URL redirection - navigate from short URL to long URL/FQDN
3. Link Analytics - track number of times each short URL is accessed (stretch)

4. 1-5B total lifetime URLs
